Remediation Recommendations

  • Set the HttpOnly flag on all session cookies to prevent XSS attacks
  • Set the Secure flag on cookies when using HTTPS
  • Set the SameSite attribute to 'Strict' or 'Lax' to prevent CSRF attacks
  • Use appropriate Domain and Path attributes to limit cookie scope
  • Ensure session cookies are set as session cookies (no persistent storage)
  • Regularly audit cookie attributes during security reviews
